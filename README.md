# 🎨 Anime Meme Maker (极简动漫表情包生成器)

欢迎来到 **Anime Meme Maker**！这是一个使用 Python 编写的轻量级工具。只需要上传一张动漫截图，再输入你想说的话，它就能瞬间帮你生成带有经典“黑边白字”风格的表情包。

## 📁 核心文件说明
在本项目中，你只需要关心这四个文件：
* `main.py`：服务的大门（API 接口），负责接收你的图片和文字。
* `generator.py`：幕后修图师，负责把文字完美地画到图片上。
* `requirements.txt`：项目清单，告诉电脑需要安装哪些基础工具。
* `font.ttf`：**（需自行准备）** 字体文件，决定了你表情包上的字长什么样。推荐使用粗体（如黑体）。

---

## 🚀 如何在本地运行？ (小白无脑版)

### 第一步：准备环境
1. 确保你的电脑上安装了 **Python**（建议版本 3.9 以上）。
2. 下载本项目所有文件到一个文件夹中。
3. **重要：** 记得自己找一个字体文件放进文件夹，并改名为 `font.ttf`！

### 第二步：安装魔法工具
打开你的命令行终端（Windows 用户搜 `cmd`，Mac 用户搜 `Terminal`），输入路径进入你的文件夹，然后执行：
```bash
pip install -r requirements.txt
```
*(这会让电脑自动下载图片处理所需的库，只需运行一次)*

### 第三步：启动服务
继续在终端里输入：
```bash
uvicorn main:app --reload
```
如果看到类似 `Application startup complete` 的绿字，恭喜你，服务启动成功了！

## 🎮 怎么使用它生成表情包？（免写代码）

我们为你准备了一个非常直观的网页操作界面！

1. 服务启动后，打开浏览器，访问：[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
2. 你会看到一个绿色的 **POST /generate/** 框框，点击它将其展开。
3. 点击右上角的 **"Try it out"** (试一试) 按钮。
4. 接下来就是填空题了：
   * **image**: 点击“选择文件”，上传一张你的动漫截图。
   * **top_text**: 输入你想在图片顶部显示的文字（比如：战斗力只有5）。
   * **bottom_text**: 输入你想在图片底部显示的文字（比如：的渣渣）。
5. 点击巨大的蓝色 **"Execute"** (执行) 按钮。
6. 往下滚动一点，在 "Responses" 区域你就会看到生成好的表情包啦！直接右键“图片另存为”即可！
